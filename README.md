# PS4HEN v2.1.1

## Features
- Homebrew Enabler
- Jailbreak
- Sandbox Escape
- Debug Settings
- External HDD Support
- VR Support
- Remote Package Install
- sys_dynlib_dlsym patchs
- deci_tty_write hook
- MMAP Patches
- Never Disable Screenshot ShellCore Patch
- Enables UART
- Rest Mode Support (may have been broken again)
